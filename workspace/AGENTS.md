# Nova OpenCode Workspace

This is the default workspace for the nova-opencode AI coding assistant.
Files created and modified by OpenCode sessions are stored here.

## Available Tools

OpenCode has access to the following MCP servers for interacting with the
Nova robotics platform:

- **robotdrive** — Robot motion planning and control
- **nova_vlm** — Vision/language model for camera stream analysis

## Pre-installed CLI Tools

- `git` — Version control
- `python3` — Python interpreter
- `nova` — Wandelbots Nova CLI for platform management
